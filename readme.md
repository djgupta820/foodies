# Foodies
A simple food ordering web application where one can be registered as reatiler to sell his/her own product or can be registered as consumer where he/she can order.

## Technologies Used: 
<ul> 
    <li> 
        <ul> 
            FrontEnd: 
            <li> EJS </li>
            <li> CSS </li>
            <li> JavaScript </li>
            <li> BootStrap </li>
        </ul>
    </li>
    <li> 
        <ul> 
            Backend: 
            <li> NodeJS </li>
            <li> ExpressJS </li>
            <li> Database: MongoDB </li>
        </ul>
    </li>
</ul>
